## 📂 Projects

### ✅ Simple Todo App
A basic task management app built with Angular.  
Features:
- Add and remove tasks
- Mark tasks as completed / pending
- Styled with SCSS
- FontAwesome icons for actions

Run it locally:
```bash
cd simple-todo-app
npm install
ng serve -o
```
